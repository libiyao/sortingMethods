# This repo includes several sorting methods such as merge sort, quick sort, and etc.

Runtimes:

                  Best Case      Worst Case     Space 
            
	Merge sort:    O(nlogn)       O(nlogn)      O(n) 


	Quick sort:    O(nlogn)       O(n^2)        O(1) 
	
	
	Selection sort:  O(n^2)       O(n^2)        O(1)
